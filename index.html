Aesir Helm Charts

Official Helm chart repository for the Aesir platform.

Usage

helm repo add aesir https://aesir-ie.github.io/helm-charts
helm repo update
helm search repo aesir

Available charts

  • aesir-agent — Aesir OTel fleet agent (manages OpenTelemetry Collectors over OpAMP).

Source and documentation: github.com/Aesir-IE/helm-charts
